## 关键决策

- mpv 优先于 ffplay/VLC，因为 mpv 提供稳定的 JSON IPC。
- 无 mpv 时继续使用现有重启式控制，避免破坏兼容性。
- IPC 使用每次播放唯一的 Windows named pipe，命令使用递增 request_id。
- 测试通过依赖注入模拟进程与 IPC，不要求开发机安装 mpv。
